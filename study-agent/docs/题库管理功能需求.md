# 题库管理功能需求文档

## 1. 概述

### 1.1 功能目标

为后台管理员提供题目录入和管理功能，支持题目的增删改查、批量导入导出，以及与知识点的关联管理。

### 1.2 相关表结构

#### t_question 题目表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键ID |
| type | VARCHAR(10) | 题目类型：选择/填空/解答 |
| content | TEXT | 题目内容（选择题不包含选项） |
| options | TEXT | 选择题选项，JSON格式如：["A. xxx","B. xxx"] |
| answer | TEXT | 正确答案 |
| analysis | TEXT | 题目解析 |
| subject_id | BIGINT | 所属学科ID |
| knowledge_point_ids | TEXT | 关联知识点ID，多个用逗号分隔 |
| difficulty | INT | 难度等级：1-5星 |
| frequency | INT | 出题频率 |
| del | TINYINT | 删除标记：0-未删除，1-已删除 |
| create_time | DATETIME | 创建时间 |
| modify_time | DATETIME | 修改时间 |

#### t_knowledge_point 知识点表（参考）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键ID |
| name | VARCHAR(100) | 知识点名称 |
| code | VARCHAR(50) | 知识点编码 |
| parent_id | BIGINT | 父知识点ID，0表示顶级 |
| subject_id | BIGINT | 所属学科ID |
| grade_id | BIGINT | 所属年级ID |
| difficulty | INT | 难度等级：1-5星 |
| description | VARCHAR(500) | 知识点描述 |
| sort_order | INT | 排序顺序 |

## 2. 功能需求

### 2.1 题库列表页

**路由**: `/admin/question`

**功能描述**:
- 分页展示所有题目
- 支持多条件筛选和关键词搜索

**筛选条件**:
| 条件 | 类型 | 说明 |
|------|------|------|
| 学科 | 下拉选择 | 关联 t_subject 表 |
| 知识点 | 级联选择 | 根据学科联动，支持多选 |
| 题目类型 | 下拉选择 | 选择/填空/解答 |
| 难度等级 | 下拉选择 | 1-5星 |
| 关键词 | 文本输入 | 搜索题目内容 |

**列表字段**:
| 字段 | 说明 |
|------|------|
| ID | 题目ID |
| 题目内容 | 显示题目内容摘要（超长截断） |
| 类型 | 题目类型标签 |
| 学科 | 所属学科名称 |
| 知识点 | 关联知识点名称（最多显示3个） |
| 难度 | 难度星级展示 |
| 出题频率 | 数值 |
| 创建时间 | 格式：YYYY-MM-DD HH:mm |
| 操作 | 编辑/删除 |

**分页**:
- 默认每页20条
- 支持切换每页条数（10/20/50/100）

### 2.2 题目录入页

**路由**: `/admin/question/add`

**功能描述**:
- 单个题目录入
- 支持选择题、填空题、解答题三种类型

**表单字段**:

| 字段 | 类型 | 必填 | 验证规则 |
|------|------|------|----------|
| 题目类型 | 单选按钮 | 是 | 选择/填空/解答 |
| 题目内容 | 富文本编辑器 | 是 | 最大5000字符 |
| 选项 | 多行文本（选择题必填） | 条件必填 | 每行一个选项，格式：A. xxx |
| 正确答案 | 文本 | 是 | 选择题为选项字母，填空/解答为文本 |
| 题目解析 | 富文本编辑器 | 否 | 最大3000字符 |
| 所属学科 | 级联选择器 | 是 | 联动知识点选择 |
| 关联知识点 | 多选树形选择器 | 是 | 必选至少一个 |
| 难度等级 | 星级评分 | 是 | 1-5星 |

**交互要求**:
- 题目类型切换时，动态显示/隐藏选项输入框
- 学科切换时，知识点选择器联动更新
- 录入成功后跳转到列表页

### 2.3 题目编辑页

**路由**: `/admin/question/edit/:id`

**功能描述**:
- 编辑已有题目
- 页面结构与录入页一致
- 回显原有数据

**特殊处理**:
- 加载时获取题目完整信息
- 保存时保留创建时间和创建人

### 2.4 题目删除

**删除类型**:
1. **单个删除**: 列表操作列删除按钮
2. **批量删除**: 列表勾选后批量删除

**交互要求**:
- 删除前弹出确认对话框
- 删除后刷新列表并显示成功提示

### 2.5 批量导入

**路由**: `/admin/question/import`

**功能描述**:
- 支持 Excel 文件导入题目

**导入流程**:
1. 下载导入模板
2. 上传 Excel 文件
3. 预览导入数据
4. 确认导入

**Excel 模板字段**:
| 列名 | 说明 | 必填 |
|------|------|------|
| type | 题目类型 | 是 |
| content | 题目内容 | 是 |
| options | 选项（选择题填写） | 否 |
| answer | 正确答案 | 是 |
| analysis | 解析 | 否 |
| subject_id | 学科ID | 是 |
| knowledge_point_ids | 知识点ID（逗号分隔） | 是 |
| difficulty | 难度(1-5) | 是 |

**错误处理**:
- 显示导入失败行号和错误原因
- 支持部分成功导入（失败行跳过）

### 2.6 批量导出

**功能描述**:
- 导出筛选后的题目列表为 Excel

**导出字段**:
- 与列表字段一致
- 包含所有筛选结果（上限10000条）

### 2.7 知识点管理入口

**功能描述**:
- 在题库管理页面提供知识点管理入口
- 跳转至 `/admin/knowledge-point`

**知识点管理页面** (`/admin/knowledge-point`):

| 功能 | 说明 |
|------|------|
| 知识点列表 | 树形展示知识点结构 |
| 新增知识点 | 支持选择父节点 |
| 编辑知识点 | 修改知识点信息 |
| 删除知识点 | 逻辑删除，需确认 |

## 3. API 接口设计

### 3.1 题目相关接口

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 题目列表 | GET | /admin/question/list | 分页查询 |
| 题目详情 | GET | /admin/question/{id} | 获取单个题目 |
| 新增题目 | POST | /admin/question | 创建题目 |
| 更新题目 | PUT | /admin/question | 更新题目 |
| 删除题目 | DELETE | /admin/question/{id} | 删除单个题目 |
| 批量删除 | DELETE | /admin/question/batch | 批量删除 |
| 导入题目 | POST | /admin/question/import | Excel导入 |
| 导出题目 | GET | /admin/question/export | Excel导出 |

### 3.2 知识点相关接口

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 知识点树 | GET | /admin/knowledge-point/tree | 获取知识点树 |
| 知识点列表 | GET | /admin/knowledge-point/list | 分页查询 |
| 知识点详情 | GET | /admin/knowledge-point/{id} | 获取单个知识点 |
| 新增知识点 | POST | /admin/knowledge-point | 创建知识点 |
| 更新知识点 | PUT | /admin/knowledge-point | 更新知识点 |
| 删除知识点 | DELETE | /admin/knowledge-point/{id} | 删除知识点 |

## 6. 验收标准

### 6.1 功能验收

- [ ] 题目列表分页、筛选、搜索正常
- [ ] 题目录入表单验证完整
- [ ] 题目编辑保留原始创建时间
- [ ] 单个/批量删除正常
- [ ] Excel导入支持预览和错误提示
- [ ] Excel导出数据完整
- [ ] 知识点树形展示正确
- [ ] 知识点CRUD功能正常

### 6.2 交互验收

- [ ] 学科切换时知识点联动
- [ ] 题目类型切换时表单动态变化
- [ ] 删除操作有确认提示
- [ ] 操作成功/失败有明确提示
- [ ] 加载状态有loading提示

### 6.3 性能要求

- 列表查询响应时间 < 500ms
- 单个题目保存响应时间 < 200ms
- 批量导入1000条 < 30s
