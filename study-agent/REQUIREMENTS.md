# 学习辅助智能体系统 - 需求规格说明书

## 1. 项目概述

### 1.1 项目名称
**Study Agent（学习辅助智能体）**

### 1.2 项目愿景
让学习像玩游戏一样上瘾的 K12 学习辅助系统。

### 1.3 核心价值
- 错题不只告诉"错了"，而是追踪到具体知识点薄弱点
- 练习题像游戏副本通关，做对题目获得奖励和成就感
- 学生能看到自己的"知识地图"和成长轨迹

### 1.4 目标用户
- **小学生/初中生/高中生**：主要学习用户
- **家长**：监督学习进度
- **老师**：布置作业、查看学情报告

## 3. 功能需求

### 3.1 学生与账号管理

#### 3.1.1 学生注册
- 用户名（唯一）、密码、昵称、年级
- 注册后初始金币/钻石/经验值均为 0，等级为 1

#### 3.1.2 学生登录
- 用户名 + 密码登录
- 返回学生基本信息（JWT 令牌，规划中）

#### 3.1.3 学生信息管理
- 查看/编辑个人信息
- 头像、昵称修改
- 所属年级显示

### 3.2 年级与学科管理

#### 3.2.1 年级管理
- 支持学段：小学、初中、高中
- 预置 12 个年级：小学 1-6 年级，初中 1-3 年级，高中 1-3 年级

#### 3.2.2 学科管理
- 每个年级包含：语文、数学、英语
- 支持多教材版本（人教版等）

### 3.3 知识点体系

#### 3.3.1 知识点树形结构
```
学科
└── 章节
    └── 知识点
        └── 子知识点
```

#### 3.3.2 知识点属性

|------|------|
| name | 知识点名称 |

- **小学三年级英语下学期**：6 个单元、语法汇总（30 个知识点）

### 3.4 题库管理

#### 3.4.1 题目属性
| 属性 | 说明 |
|------|------|
| type | 题目类型：选择/填空/解答 |

#### 3.4.2 已初始化的题目

### 3.5 AI 错题分析

#### 3.5.1 核心能力
- 调用 LLM（OpenAI）分析错因
- 返回结构化分析结果

#### 3.5.2 分析维度
| 错误类型 | 说明 |
|----------|------|
| 知识点缺失 | 学生未掌握该知识点 |
| 理解偏差 | 概念理解有误 |
| 审题错误 | 未正确理解题意 |
| 计算失误 | 计算过程出错 |

#### 3.5.3 分析结果
```json
{
  "knowledge_point": "知识点名称",
  "error_type": "知识点缺失",
  "mastery_level": 0.3,
  "practice_suggestion": "建议做 X 道 xxx 类型的题目"
}
```

#### 3.5.4 兜底策略
- AI 调用失败时，使用 Levenshtein 距离计算相似度
- 相似度 > 50% 判定为"理解偏差"，否则为"知识点缺失"

### 3.6 游戏化奖励系统

#### 3.6.1 虚拟货币
| 货币 | 说明 | 获取方式 |
|------|------|----------|
| 💎 钻石 | 稀有货币 | 完成挑战、连续打卡、成就奖励 |
| 🪙 金币 | 普通货币 | 答题奖励、日常任务 |
| ⭐ 经验值 | 升级用 | 答题、完成任务 |

#### 3.6.2 等级系统
- 初始等级 1 级，经验值 0
- 每级所需经验值 = 等级 × 100
- 升级时扣除对应经验值，等级 +1

#### 3.6.3 奖励计算公式
```
基础奖励 = 10 金币 + 5 经验值
连击加成 = 1 + (连击次数 × 0.2)   // 每连击 +20%
难度加成 = 1 + (难度星级 × 0.1)   // 每星难度 +10%
最终奖励 = 基础奖励 × 连击加成 × 难度加成
```

#### 3.6.4 连续学习机制
- 每次学习增加连续学习天数
- 断签重置为 0

### 3.9 知识点掌握度跟踪

#### 3.9.1 掌握度计算
- 正确率 = 正确次数 / 练习次数
- 掌握度 = 正确率（0.0 - 1.0）

#### 3.9.2 星级评定
| 正确率范围 | 星级 |
|------------|------|
| ≥ 90% | 5 星 |
| ≥ 80% | 4 星 |
| ≥ 60% | 3 星 |
| ≥ 40% | 2 星 |
| ≥ 20% | 1 星 |
| < 20% | 0 星 |

### 3.11 冲刺模式

> **详细设计文档**：[docs/SPRINT_MODE_DESIGN.md](./docs/SPRINT_MODE_DESIGN.md)

#### 3.11.1 核心定义
**🏃 冲刺模式（Sprint Mode）** — 在限定时间（60秒）内，以最快速度、最高准确率完成尽可能多的题目。

#### 3.11.2 核心特性
| 特性 | 说明 |
|------|------|
| ⏱️ 限时挑战 | 每轮 60 秒固定时间 |
| 🔥 连击机制 | 连续答对加速奖励累积 |
| 📊 实时统计 | 显示当前得分、剩余时间、连击数 |
| 🏆 成绩排名 | 记录历史最佳成绩 |

#### 3.11.3 计分规则
```
单题得分 = 100 × 时间加成系数 × 连击加成系数

时间加成：
  - 0-30 秒：× 1.0
  - 30-45 秒：× 1.2
  - 45-60 秒：× 1.5

连击加成 = 1.0 + (连击数 × 0.1)，封顶 3.0
```

#### 3.11.4 奖励规则
| 连击数 | 钻石奖励 |
|--------|----------|
| ≥ 3 连击 | +1 💎 |
| ≥ 5 连击 | +5 💎 |
| ≥ 10 连击 | +20 💎 |
| ≥ 20 连击 | +50 💎 |

#### 3.11.5 新增成就
| 成就名称 | 条件 | 奖励 |
|----------|------|------|
| 初次冲刺 | 完成首次冲刺 | 20金币, 10经验 |
| 冲刺新星 | 单轮答对5题 | 50金币, 25经验 |
| 冲刺达人 | 单轮答对10题 | 100金币, 50经验, 3钻石 |
| 完美冲刺 | 正确率100% | 200金币, 100经验, 10钻石 |
| 连击冲刺 | 单轮10连击 | 150金币, 75经验, 5钻石 |
| 2000分 | 单轮得分2000+ | 300金币, 150经验, 15钻石 |
| 3000分 | 单轮得分3000+ | 500金币, 250经验, 30钻石 |

|------|------|
